
# Cómo cambiar el logo

- Reemplaza `assets/logo.png` por tu imagen (ideal fondo transparente, cuadrada).
- Mantén también `assets/logo.ico` si quieres icono de ventana en Windows (puedes generarlo desde el PNG).
- El README ya muestra el logo centrado.
- La app intenta usar `logo.ico` y, si falla, `logo.png`.
